## 实现联系方式按钮悬浮效果

### 目标
当用户悬浮在联系方式的社交链接按钮上时，按钮上边缘向上移动，并在按钮上方出现一个容纳方形二维码的区域。

### 实现思路
1. **为每个社交链接按钮添加二维码容器**：
   - 在每个 `.social-link` 内部添加一个绝对定位的二维码容器
   - 容器初始状态为隐藏，使用 `opacity: 0` 和 `transform: translateY(10px)`
   - 容器尺寸设置为方形，显示对应平台的二维码图片

2. **修改按钮样式和交互**：
   - 确保 `.social-link` 具有 `position: relative` 属性
   - 添加悬停效果，使按钮向上移动（使用 `transform: translateY(-10px)`）
   - 同时触发二维码容器的显示动画

3. **二维码容器设计**：
   - 尺寸设置为 150x150px 的方形
   - 位于按钮正上方，底部与按钮上边缘对齐
   - 包含对应平台的二维码图片，使用 `data-image` 属性获取图片 URL
   - 添加阴影和圆角，提升视觉效果

4. **动画效果**：
   - 按钮向上移动使用平滑过渡动画
   - 二维码容器从下方滑入并淡入
   - 悬停结束时，按钮恢复原位，二维码容器滑出并淡出

### 具体实现步骤
1. **修改 CSS 样式**：
   - 更新 `.social-link` 样式，添加 `position: relative` 和过渡效果
   - 创建 `.qrcode-container` 样式，设置绝对定位、初始隐藏状态和动画效果
   - 添加 `.social-link:hover` 状态，实现按钮上移和二维码显示

2. **更新 HTML 结构**：
   - 在每个 `.social-link` 内部添加 `.qrcode-container` 元素
   - 容器内包含 `<img>` 标签，使用 `data-image` 属性作为图片源

3. **确保二维码图片正确显示**：
   - 调整二维码图片的 `object-fit` 属性，确保方形显示
   - 为图片添加适当的样式，确保清晰度

### 预期效果
- 当鼠标悬停在社交链接按钮上时，按钮平滑向上移动约 10px
- 按钮上方出现一个 150x150px 的方形二维码容器，包含对应平台的二维码
- 二维码容器从下方滑入并淡入，带有平滑过渡效果
- 悬停结束时，按钮恢复原位，二维码容器滑出并淡出
- 保持原有大二维码区域的显示，实现双重二维码展示效果

### 注意事项
- 确保二维码容器的 z-index 足够高，避免被其他元素遮挡
- 考虑响应式设计，在小屏幕上调整二维码容器的尺寸和位置
- 确保动画效果流畅，不影响页面性能
- 保持与现有设计风格的一致性
