# Furry自我介绍网站设计方案

## 1. 网站架构

### 1.1 页面结构
- **导航栏**：顶部固定，滚动时隐藏，包含进度条指示当前位置
- **英雄区域**：大型横幅展示，包含fursona主图和简短介绍
- **关于我**：个人基本信息和简介
- **我的Fursona**：详细的奶昔(Lancee)兽设信息、图片和背景故事，突出宝石一样的角（左角残破）特征
- **作品展示**：创意展廊，约8个作品，支持手作、摄影、建模、画作四种类型展示
- **兴趣爱好**：个人兴趣和社区参与
- **联系我**：QQ、X、bilibili、Github、Steam、Xbox等社交媒体链接
- **页脚**：版权信息

### 1.2 交互结构
- 平滑滚动导航
- 导航栏底部进度条随页面滚动同步移动，进度条是一个矩形长条，起点和终点分别是第一个和最后一个导航标签
- 作品卡片悬停1.02倍放大效果
- 内容区块加载动画
- 背景颜色随滚动位置动态变化，颜色的变化可以被滚动操作实时打断
- 模态框查看详细作品并支持下载功能

## 2. 设计系统

### 2.1 颜色方案（参考Apple风格）
- **主色调**：
  - 白色 (#FFFFFF) - 背景基础色
  - 浅灰 (#F5F5F7) - 区块背景
  - 中灰 (#86868B) - 次要文本
  - 深灰 (#1D1D1F) - 主要文本
  - 黑色 (#000000) - 强调文本
- **强调色**：
  - 低饱和度水蓝色 (#3A7CA5) - 默认按钮状态和主色调
  - 高饱和度青绿色 (#2A9D8F) - 按钮悬停状态和交互元素
- **支持色**：
  - 成功绿 (#34C759)
  - 警告黄 (#FF9500)
  - 错误红 (#FF3B30)
- **背景动态变化**：根据滚动位置平滑过渡不同的浅色背景，颜色的变化可以被滚动操作实时打断

### 2.2 排版系统
- **字体选择**：
  - 主要字体：Inter (Web字体) 或 SF Pro Display (Apple系统字体)
  - 备选字体：-apple-system, BlinkMacSystemFont, sans-serif
  - 字体风格：无衬线，字重较大
- **字号层级**：
  - 大标题：clamp(2.5rem, 5vw, 4rem)
  - 中标题：clamp(1.75rem, 3vw, 2.5rem)
  - 小标题：clamp(1.25rem, 2vw, 1.75rem)
  - 正文：16px，约为标题50%高度
  - 小文本：clamp(0.875rem, 1vw, 1rem)
- **行高**：
  - 标题：1.2
  - 正文：1.6（宽松大气）
- **段落间距**：宽松大气，提升阅读舒适度

### 2.3 间距系统
- **基础间距单位**：4px
- **间距倍数**：
  - 小型间距：8px (2u)
  - 中型间距：16px (4u)
  - 大型间距：24px (6u)
  - 特大间距：32px (8u)
  - 区块间距：48px (12u) 到 96px (24u)
- **统一间距系统**：
  - 图片和文本间距：使用中型间距
  - 作品卡片间距：使用中型到大型间距
  - 页面标题和内容区域内边距：使用大型到特大间距

### 2.4 阴影系统
- 轻微阴影：0 2px 8px rgba(0, 0, 0, 0.05)
- 中等阴影：0 4px 16px rgba(0, 0, 0, 0.1)
- 强烈阴影：0 8px 32px rgba(0, 0, 0, 0.15)
- 悬停效果：0 12px 32px rgba(0, 0, 0, 0.2)

### 2.5 动画系统
- **过渡时长**：
  - 快速：0.2s
  - 中等：0.3s
  - 慢速：0.5s
- **缓动函数**：
  - 基本：ease
  - 进出：cubic-bezier(0.4, 0, 0.2, 1)
  - 弹性：cubic-bezier(0.68, -0.55, 0.27, 1.55)
- **关键动画**：
  - 渐变入场
  - 视差滚动
  - 悬停放大（1.02倍）
  - 滑动切换
  - 加载动画
  - 背景颜色平滑过渡
  - 导航进度条同步滚动

## 3. 响应式设计策略

### 3.1 断点设置
- 移动设备：< 768px
- 平板设备：768px - 1024px
- 桌面设备：1024px - 1440px
- 大屏设备：> 1440px

### 3.2 布局适配
- **优先PC端优化**
- 移动设备：单列布局，简化导航为汉堡菜单，内容顺序与PC保持一致
- 平板设备：双列网格布局，精简内容
- 桌面设备：多列网格布局，完整功能展示
- 大屏设备：优化宽屏显示，增强视觉冲击力

### 3.3 组件适配
- 图像：响应式调整大小，确保清晰度
- 文本：自适应字号，保证可读性
- 导航：PC端完整显示，移动端使用汉堡菜单
- 展廊：移动端单列滚动，桌面端网格布局
- 作品分类：移动端简化为下拉选择，桌面端保持标签式布局

## 4. 特殊功能设计

### 4.1 作品展示
- **作品分类**：
  - 分类顺序：手作、摄影、建模、画作
  - 筛选方式：左侧一列标签式布局
  - 默认选中"全部"分类
  - 点击新标签弹出，其余收回，也可统一颜色样式
  - 筛选按钮点击后直接显示对应分类作品
- **作品卡片**：
  - 网格布局展示
  - 悬停效果：1.02倍轻微放大
  - 交互元素：包含下载功能
- **作品详情**：
  - 点击卡片查看大图
  - 提供下载功能

### 4.2 Fursona展示
- 主视图大图展示奶昔(Lancee)形象
- 突出宝石一样的角（左角残破）特征
- 交互式特性说明
- 背景故事整合

### 4.3 导航设计
- 顶部固定，滚动时隐藏
- 导航文本平时为白色
- 文本悬停时略微放大，逻辑类似按钮
- 底部进度条：
  - 矩形长条设计
  - 进度条随页面滚动同步移动
  - 起点和终点对应第一个和最后一个导航标签
  - 页面滚动到对应区域时，进度条移动到相应导航项下方

### 4.4 Logo设计
- 圆环设计
- 环的宽度是半径的50%
- 颜色根据背景动态调整，建议黑白灰

### 4.5 按钮设计
- **形状**：
  - 小按钮：胶囊形
  - 大按钮：圆角矩形
- **状态变化**：
  - 默认状态：低饱和水蓝色 (#3A7CA5)
  - 悬停状态：轻微放大，变成高饱和青绿色 (#2A9D8F)
  - 点击状态：回弹效果
- **圆角系统**：为全局设定统一的圆角系统

### 4.6 社交链接设计
- **展示平台**：QQ、X、bilibili、Github、Steam、Xbox等（主要面向国内用户）
- **设计要求**：图标统一颜色、大小，整齐排列
- **交互效果**：
  - 悬浮时可展示二维码
  - 点击图标跳转到社交平台主页

### 4.7 性能优化
- **图片优化**：实现懒加载，优先显示欢迎页面
- **加载策略**：减少用户等待时间

## 5. 浏览器兼容性
- **主要支持**：Safari、Edge、Chrome
- **优先优化**：PC端浏览器体验

## 5. 性能优化策略
- **图片懒加载**：优先展示欢迎页面，减少用户等待时间
- **代码分割**：按需加载组件
- **资源压缩**：优化CSS和JavaScript文件大小
- **字体优化**：使用现代字体加载策略
- **缓存策略**：合理设置缓存规则
- **初始加载优化**：确保快速的页面加载体验

---

本设计方案基于Apple网站的简洁大气风格，结合Furry文化元素，通过优雅的动画效果和精心的排版，打造一个视觉吸引力强、用户体验良好的个人介绍网站。设计采用浅色主体配合低饱和水蓝和高饱和度青绿作为点缀色，实现了根据滚动位置动态调整背景颜色的效果，同时为各类交互元素提供了精心设计的动画过渡，使整个网站既简洁又富有现代感。
